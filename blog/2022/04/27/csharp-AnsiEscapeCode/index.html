WPF 文本框仿控制台做一个五彩斑斓的黑

简介

参考 维基百科 ANSI escape code 控制台程序使用 ANSI 转义来体现文字颜色，在控制台中显示出一个五彩斑斓的黑，也有很多控制台程序用这些带文字的颜色显示自家 LOGO。
WPF 想要在一行中加载不同颜色的文字，使用富文本控件 RichTextBox 即可，但是需要先解析文本中的 Ansi 字符。

解析字符

Ansi 编码由 \u001b[ 开头，加数值编码，由 m 结尾。
数值 0 为默认值。
数值 30-37;90-97 为文字颜色。
数值 40-47;100-107 为背景颜色。
通过正则表达式 \u001b[(.*?)m 匹配文本，后续的文本即为设定颜色。
