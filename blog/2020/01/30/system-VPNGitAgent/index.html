发表于 2020-01-30 | 分类于 System

解决国内 Github 访问过慢

简介

开了 VPN 之后使用 Git 连接 github 的 clone pull push 命令依旧很慢，是由于 git 默认不使用代理导致，配置 git 代理后可提升速度。
